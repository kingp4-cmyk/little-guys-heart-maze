# Little Guys: Heart Maze

A stylized 3D maze game developed with Godot Engine and Blender.

> **Development Status:** In Progress

## About the Game

Little Guys: Heart Maze is a third-person maze game in which the player explores a maze, finds the missing Little Guys, and reaches the exit before time runs out.

The project focuses on game programming, 3D asset creation, level design, user interface development, and gameplay testing.

## Planned Features

- Third-person character movement
- Camera controls
- Custom 3D maze environment
- Collectible Little Guy characters
- Countdown timer
- Collectible counter
- Win and game-over conditions
- Restart and main-menu systems
- Pause menu
- Original 3D assets created in Blender
- Sound effects and background music
